@PURPOSE: Provide basic statistical computations.

@MNEMONIC: Basic Development Library STATistics (bdlstat)

@DESCRIPTION: The 'bdlstat' package provides basic statisticalcomputations.  At
 the moment, this package contains a component, 'bdlstat_moment', for
 calculating mean, variance, skew, and kurtosis.
